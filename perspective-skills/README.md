# 補蒸餾的思想家視角

補上一開始建議、但一直沒回頭做的兩個視角——填你視角庫的兩個大缺口：**關係/溝通** 與 **自助/勇氣**。

| skill | 補的缺口 | 一句話 |
|-------|----------|--------|
| [`adler-perspective`](adler-perspective/SKILL.md) | 自助/勇氣/人際 | 目的論、課題分離、社會情懷（被討厭的勇氣） |
| [`nvc-perspective`](nvc-perspective/SKILL.md) | 關係/溝通 | 觀察→感受→需要→請求（把指責翻成需要） |

## 怎麼裝
和你其他 perspective skill 一樣：把資料夾放進你的 skills 目錄（或用 `skill-creator` / 女娲匯入），
之後對話裡出現觸發詞（課題分離、非暴力溝通、需要、勇氣…）就會啟用。

## 為什麼是這兩個
你的視角庫偏東方 + 深度心理 + 存在主義（老莊、榮格、Tolle…），
但「**關係怎麼溝通**」和「**華人自助爆款（阿德勒）**」這兩塊本來是空的——
療癒/覺醒內容最常被問的就是這兩類，補起來直接能產選題。

> 若你想要女娲版（更深的研究蒸餾流程），也可以在你本機說「女娲 蒸餾 阿德勒」再生成一版比對。
