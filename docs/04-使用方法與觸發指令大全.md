> 58 個 skills + 4 個模組工具的完整使用手冊:怎麼裝、怎麼觸發、每個 skill 照抄即用的範例指令。

## 一、使用方法(三種)

### 方法 1:安裝成 plugin(一次性,推薦)

在 Claude Code 輸入:

```
/plugin marketplace add xoxonco-ai/my-frist-project
/plugin install marketing-skills@marketingskills
```

之後在**任何專案**,說到觸發詞 skill 就自動啟用,不用手動開。

### 方法 2:自然語句觸發(日常主要用法)

Skill 靠「語意比對」自動啟用——你不用背指令,說白話就行,句子裡帶到觸發詞即可:

- 「幫我寫一篇 **Threads 貼文**」→ 自動啟用 `social`
- 「用**課題分離**給我選題」→ 自動啟用 `adler-perspective`
- 「這頁**轉換率**好低」→ 自動啟用 `cro`

### 方法 3:直接點名(最精準)

不確定會不會觸發時,直接叫名字:

- 「**用 copywriting skill** 幫我寫首頁文案」
- 「**用 satir-perspective** 的冰山理論分析這則留言」

### 模組工具(`modules/`)另外用指令跑

```bash
python modules/2-analytics-dashboard/dashboard.py     # 成效儀表板
python modules/3-multiplatform-rewriter/rewrite.py    # 一稿轉多平台
python modules/4-content-research/hashtag_research.py # 標籤/選題研究
python modules/5-auto-reply/app.py                    # 留言自動回覆
```

或偷懶直接說:「幫我跑多平台改寫」「跑一下標籤研究」,Claude 會代為執行。

## 十、串成工作流(組合技)

單一 skill 是零件,串起來才是產線(完整節奏見 docs/03):

```
週一:「從上週留言找 3 個卡點」(customer-research)
     →「用薩提爾+佩瑪各給 3 個選題」(視角庫)
     →「排成本週日曆」(content-strategy)

每天:「用今天選題寫 Threads,痛點直球開頭」(social)
     →「照 docs/02 規格預檢打分」
     → 跑 rewrite.py 轉多平台 → 圖卡 → 發布

週五:「本週成效總結,哪些該加倍?」(analytics)
     →「下週開一組鉤子 A/B 測試」(ab-testing)
```

**萬用起手式**:不知道叫誰時,直接描述狀況——「我的貼文都沒人看」「我想開始賣課」——對的 skill 會自己出現。
