# 流量演算強化規格（給 social-post / social-cards 用）

> 用法：把下面兩段規格，分別貼進你本機 `social-post` 和 `social-cards` 這兩個 skill 的
> instructions／SKILL.md 裡。之後每次發文，skill 會**自動照這份規則做預檢**，不合格就先提醒你改。
>
> 這份規格內建你「浮生矩陣」的品牌邊界：療癒、`妳`-語感、anti-hype（不恐嚇／不神化／不絕對預測／不成功學／不顯化神話）。

---

## A. `social-post` 發文前預檢規格

在真正送出前（在你回「確認」之前），先對這篇貼文逐項打分，把不到標的地方列出來給我改：

### 1. Hook 強度（前 1 行 / 前 3 秒）— 佔比最重
逐項檢查第一句是否命中至少一種鉤子：
- [ ] **痛點直球**（說中對方一個沒被說破的狀態）
- [ ] **反常識**（「X 不是你以為的那樣」）
- [ ] **身份對號**（「如果你也是那種…的人」）
- [ ] **懸念**（留一個要看下去才知道的缺口）

評分：命中 0 種 → ❌ 擋下要求重寫；命中 1 種 → ⚠️ 可發但建議加強；命中 2 種以上 → ✅

### 2. 平台適配（依目標平台切換規則）
| 平台 | 必檢項 |
|------|--------|
| Threads | 第一句可獨立成立；結尾有開放問題或關鍵字 CTA；≤ 100 字更佳 |
| IG caption | 第一行（被摺疊前）就是鉤子；結尾有「留言關鍵字」CTA |
| Facebook | 第一人稱敘事；文末催長留言（你也是嗎／二選一）；不放站外連結 |

### 3. 儲存 / 收藏誘因（IG 專檢）
- [ ] 內容含「可收藏」元素（清單、練習步驟、可對照的自我檢查）→ 沒有就提醒補一個

### 4. CTA 與名單串接
- [ ] CTA 關鍵字存在，且該關鍵字**已在模組 5 的 `keyword_map.csv` 裡有對應練習連結**
- [ ] 若關鍵字不在對照表 → ⚠️ 提醒「這個關鍵字還沒接自動私訊，留言的人會收不到練習」

### 5. 標籤（IG / 小紅書）
- [ ] 3–5 個精準療癒 tag（非 30 個泛 tag）

### 6. 品牌紅線（任一命中就 ❌ 擋下）
- [ ] 有無恐嚇式語句（「不改變你就完了」）
- [ ] 有無神化 / 絕對預測（「一定會」「注定」）
- [ ] 有無成功學 / 顯化神話（「吸引力法則保證…」）
- [ ] 有無把心理 / 醫療 / 財務問題包裝成靈性唯一解
- [ ] 有無醫療療效宣稱（「治好憂鬱」）

### 7. 發布時段提醒
- 對照建議時段（Threads 早 7–9 / IG 晚 21–23 睡前）；偏離就提示，但不強制。

**輸出格式**：預檢完先印一張小表（各項 ✅/⚠️/❌ + 一句原因），再問我是否「確認」送出。

## B. `social-cards` 封面／輪播圖優化規格

依【輪播說明】做 7 張圖時，額外套用：

### 第 1 張（封面）＝ 點擊率決勝點，佔小紅書/IG 觸及 70%
- [ ] **主字級要夠大**：一眼可讀的一句話（金句或痛點），字數 ≤ 14
- [ ] **含情緒關鍵字**：療癒 / 內耗 / 焦慮 / 放下 / 保護…（利於搜尋與對號）
- [ ] **對比清楚**：深色字 or 高對比底，縮圖尺寸下仍清楚
- [ ] 避免整張都是小字或滿版裝飾把重點蓋掉

### 第 2–6 張（內容）
- [ ] 一張一個重點，不要一張塞滿
- [ ] 行距鬆、留白夠；手機上不擠

### 第 7 張（收尾＝轉換）
- [ ] 放「今日練習題目 + 留言關鍵字 CTA」
- [ ] 關鍵字與 `keyword_map.csv` 一致（確保留言的人收得到練習）

### 通用
- [ ] 文字不溢出、不被裁切、不遮住主視覺
- [ ] 全套 7 張風格一致（同色系、同字體、同版式）→ 建立品牌識別

**輸出**：先給 7 張預覽 + 標出「封面是否通過 CTR 檢查」，再問是否輸出正式檔。

## 一句話總結
`social-post` 管「發出去的每篇都夠格被演算法推」，`social-cards` 管「第一張圖就讓人想點」。
兩者的 CTA 關鍵字都回扣到**模組 5** 的自動私訊，讓「流量 → 名單」不漏接。
