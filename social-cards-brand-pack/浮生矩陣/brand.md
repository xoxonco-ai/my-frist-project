# Brand Pack · 浮生矩陣（療癒・覺醒）

> 這是給 **social-cards-engine** 用的 brand pack（品牌插件）。
> 用法：把整個 `浮生矩陣/` 資料夾複製到 social-cards-engine 的 `brands/` 底下，
> 之後說「用 social-cards 幫我做圖卡，品牌用浮生矩陣」即可載入本 pack。
> 觸發：你說「浮生矩陣風格」「療癒風圖卡」「妳的品牌」→ 載入本 pack。

## 🎨 視覺識別（兩個方向，擇一或並用）

> ⚠️ 修正：最早的奶油/綠色票是取自你的**後台儀表板 HTML**，不是圖卡本體。
> 你實際在跑的圖卡是**黑金明體**。以下提供兩版，`render_preview.py` 兩版都已實作。

### 方向 A · 黑金（對齊你目前的實際圖卡）
```css
--bg:#0b0a08;        /* 近黑底 + 燙金放射/木紋質感 */
--ink:#f4efe6;       /* 主字近白 */
--gold:#d8b24a;      /* 燙金 accent（關鍵詞/第二行標題）*/
--box-bd:rgba(201,162,39,.38);  /* 金色細框 */
--kicker:#c9a227;    /* 房間標籤金 */
```
字體：明體/宋體大字（Noto Serif TC / 思源宋體）＝貴氣、有力量。

### 方向 B · 溫柔（更貼療癒 anti-hype 的替代版）
```css
--bg:#faf7f2;      /* 奶油米白底（留白、不刺眼）*/
--card:#ffffff;
--ink:#3a352f;     /* 暖褐主字 */
--acc:#5c7d5c;     /* 鼠尾草綠 accent */
--accbg:#eef3ee;   /* 淺綠練習框 */
--line:#e8e2d9;
```
字體：PingFang TC / Noto Sans TC，行距鬆、留白多。

> 兩版共用同一套結構（房間標籤 + 標題框 + 內文框 + 頁尾 wordmark），
> 差別只在色票與字體。實作見同層 `../render_preview.py`。

- **簽名母題**：
  - 大量**留白**與**呼吸感**（療癒品牌的核心體感，不塞滿）。
  - 重點句用 `--accbg` 淺綠底做**柔性 highlight**（不用黃黑紅那種衝突色）。
  - 每張左上或頁尾放**房間標籤**（見敘事 DNA），像「見幻筆記 · 03」。
  - 頁尾一葉 🌿 或細線 wordmark「浮生矩陣」。
- **字體**：`"PingFang TC","Noto Sans TC",sans-serif`；標題可用較細的字重（療癒＝不張牙舞爪），行距鬆（line-height ≥ 1.8）。
- **尺寸**：**4:5（1080×1350）**滿版；2× 高清（`deviceScaleFactor:2`）。

## 🌿 敘事 DNA（＝ carousel-joker 的量尺，產文時就自帶）

- **語感**：第二人稱「**妳**」，像寫給一個具體的人，溫柔、不說教、陪伴而非指導。
- **句子**：短句、留白、有停頓感；用「也許」「先別急著」這類鬆綁詞，不下命令。
- **結構母題**：`痛點對號 → 翻轉（這不是缺陷/錯，是…）→ 一個可做的小練習`。
- **房間（column）**：每篇歸一個房間，圖卡標籤沿用——
  `木雕哲學`｜`見幻筆記`｜`浮生解碼`｜`回家`｜`登出指南`。
- **anti-hype 硬邊界（任一命中 → carousel-joker 直接 FAIL）**：
  - ❌ 恐嚇（「不改變妳就完了」）
  - ❌ 神化 / 絕對預測（「一定會」「注定」「顯化保證」）
  - ❌ 成功學 / 顯化神話
  - ❌ 把心理/醫療/財務問題包成靈性唯一解
  - ❌ 醫療療效宣稱（「治好憂鬱」）
  - ✅ 用「陪伴、覺察、練習、也許、妳可以」的語氣

## 🎯 封面（第 1 張）＋ CTA（sends / saves 優先）

- **封面**：一句金句大字（≤ 14 字），含情緒關鍵字（內耗／放下／敏感／保護…）撐開好奇缺口。
  - 例：「洞不是缺陷，而是入口」「妳不是太敏感，是太早學會偵測危險」。
  - 綠 accent 只點在**一個關鍵詞**上，其餘暖褐，維持療癒的安靜感。
- **第 2–6 張**：一張一個重點，鬆排、留白；練習步驟用 `--accbg` 綠框。
- **第 7 張（轉換）**：今日練習題 + **留言關鍵字 CTA**。
  - CTA 句型走 social-post 的 R35/R39：
    - saves：「**存起來**，晚上做一次自我檢查 🌿」
    - sends：「**傳給那個總是把自己放最後的人** 🌿」
    - keyword：「留言『**入口**』，我寄妳完整書寫練習」
  - ⚠️ **關鍵字必須與模組 5 `keyword_map.csv` 一致**，留言的人才收得到自動私訊。

## 🤝 與其他 skill 的接法

- **social-post**：擴散權重（sends>saves>likes）、發文、學語氣的**單一事實源**——圖卡 CTA 一律對齊它的 R 規則。
- **模組 5（本 repo）**：第 7 張的關鍵字 → 官方 API 自動私訊練習連結 + 收名單。
- **typo-studio**：若要手動微調中文避頭尾/字級，可把文字丟進去排版後再出圖。
- **ai-media-generator**：需要封面配圖/材質底時借它生圖（療癒、柔光、慢節奏）。

## ✅ 過閘

產完照 **`carousel-joker`** skill 跑 PASS/FAIL（知識正確 / Hook / 擴散 / 品牌安全），
其中「品牌安全」套用上方 anti-hype 硬邊界。美感再過 `huashu-design`（置中/留白/不跑版）。

## 🕐 何時用本 pack
- 你所有「浮生矩陣」IG／Threads 輪播圖卡。
- 迷因風、衝突色、hard-sell 一律**不套**本 pack（那不是這個品牌）。
