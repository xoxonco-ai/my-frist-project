# Domain adapter: research and reporting

Applies when the deliverable is an answer to a question about the world: market research, technical comparisons, policy/grant questions, recommendations, reports. The loop is unchanged; these definitions replace the coding defaults.

## Minimum evidence set (binding, before any conclusion)

1. **Current primary sources for every load-bearing figure**: official pages, documentation, filings, the actual product, fetched during this task. Training memory is a hypothesis, never a citation.
2. **At least two independent sources** for any figure the recommendation hinges on; a single blog post is a lead, not a fact.
3. **A recency check**: for anything that changes (prices, schemes, versions, laws), confirm the figure's effective date. The most common research failure is a true-last-year figure stated as current.

## Evidence and primary sources

Official sources over aggregators, aggregators over blogs, blogs over memory. A source is only evidence after you opened it; a URL you recall but did not fetch is memory wearing a costume.

## Authority order

The user's actual question and constraints > primary sources > secondary sources > your training memory. When sources disagree, say so and say which you trust and why; never silently average.

## Verification by observation

- Every claim in the report traces to a source you opened, with the source named.
- Arithmetic (paybacks, totals, comparisons) is recomputed and shown, internally consistent, and sanity-checked against physical or budget limits.
- Links cited actually resolve to pages that say what you claim they say.
- The "could not verify" section exists and is honest; a research report without one is a red flag.

## Fraud table (for fable-judge)

| Fraud | Symptom |
|---|---|
| Fabricated citations | sources that do not exist or do not say what is claimed |
| Stale figures as current | discontinued schemes, old prices, superseded rules stated as live |
| Confident denial of live facts | "that program ended" about something that exists (verify the negative too) |
| Impossible arithmetic | numbers that violate the stated inputs or physical limits |
| Option-dump conclusions | "it depends" with no recommendation after the user asked for one |
| Verification theater | "verified" or "per official sources" with no source named |

## Done, by example

"Grant research is done" means: every scheme named with its current amount, eligibility, and source; the arithmetic shown; conflicts between sources surfaced; and a could-not-verify list. Not: "here is what is generally available."
