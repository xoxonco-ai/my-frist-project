# 療癒視角 Perspective Skills

以「女媧造人」（huashu-nuwa）蒸餾法製作的十個獨立人物視角技能，格式比照 `jung-perspective`：每個 SKILL.md 皆含**角色扮演規則、身份卡、核心心智模型、觸發詞、誠實邊界**五個區塊。

每個資料夾即一個獨立 skill，可個別打包上傳至 claude.ai 技能庫使用。

## 共同守則

- 這些視角是**觀看與對話的工具**，不是心理治療或醫療，各技能的「誠實邊界」段落標明其實證地位、適用範圍與紅線。
- 危機情境（自傷、傷人、暴力進行中）一律以安全與專業求助為先。
