## 二、浮生矩陣內容工作流

核心迭圈:**選題 → 產文 → 預檢 → 多平台 → 圖卡 → 發布 → 互動 → 復盤**。
每一步都標了「跟 agent 說什麼」就能觼發。

### 第 0 步(一次性):建立品牌脈絡

```
跟 agent 說:「幫我建立 product-marketing 品牌脈絡文件」
```

用 `product-marketing` 生成 `.agents/product-marketing.md`,寫入:浮生矩陣的定位、`妳` 語感、anti-hype 邆界(不恐噇/不神化/不成功學)、受眾樨跌。**之後所有 skill 都會先讀這份**,不用每次重貫。

### 每週一・選題日(30 分鐘)

1. **挖需求**:`customer-research` —
   「幫我從上週留言和私訊裡,找出大家最常卡住的 3 個主題」
2. **產選娑**:料角度 —
   「用阿德勒的課題分離 + Rogers 的被聽見,各給我 3 個本週選題」
   (12 個視角輷流用,一週配 2 個視角,6 週不重複)
3. **排日曆**:`content-strategy` + `social` —
   「把這 6 個選題排成本週內容日斩:IG Reels x2、Threads x3、小紅書 x2－
4. **查標籤**:`modules/4-content-research` — 跑 `hashtag_research.py` 補關鍵字

### 每日・產文迭圈(每篇 15 分鐘)

1. **寫初稯**:`social` + `copywriting` —
   「用今天的選題寫一篇 Threads 貼文,開頭用痛點直球鉤子」
   (開頭公式見 docs/01 第二節)
2. **預檢**:照 **docs/02 規格**逐項打分 — Hook 強度、平台適配、品牌邊界。不合格先改再發
3. **一稿多平台**:`modules/3-multiplatform-rewriter` — 跑 `rewrite.py` 轉成 IG caption、FB、小紅書癈本
4. **做圖卡**:`social-cards-brand-pack` 的 render 腳本;需要新圖用 `image`
5. **發布**:各平台購文(黃金時段見 docs/01)

### 每日・互動(10 分鐘)

- `modules/5-auto-reply` 處理留言關鍵字自動回覆(keyword_map.csv 維護觀發時發時發)
- `social` 社群聆聽 — 「找出今天值得我親自回的 5 則留言」

### 每週五・復盤日(20 分鐘)

1. **看數據**:`modules/2-analytics-dashboard` 跑 dashboard.py + `analytics` —
   「本週哪 2 篇完讀率最高?哪些選題該加倍做?」
2. **開實驗**:`ab-testing` —
   「下週用同一選題測兩種艤子:痛點直球 vs 反常識」
3. **回寫選題庫**表現好的視角/主題,下週一優先排

### 每月・成長線(名單自我現,選做)

| 階段 | Skill | 動作 |
|------|-------|------|
| 名單 | `lead-magnets` + `popups` | 做一份「情緒急救檢核表」換 email/LINE |
| 滋養 | `emails` | 7 天歡迎序刖(每天一個視角小練習) |
| 變揸 | `offers` + `pricing` + `launch` | 課程/陪伴方案的 offer 設計與上市 |
| 放大 | `referrals` + `community-marketing` | 鄵碣推薦、社群飛輸 |

### 逹階:䮤它自己跑

空宛跑 4 週後，用 `marketing-loops` 把「週一選題、每日預檢、週五復盤」設成定期自動迴圈:
「幫我把這個工作流設成 marketing loop,每週一早上自動產選題給我確認」。

## 三、速查:遇到 X 就叫 Y

| 情境 | 說這句 → 觸發 |
|------|--------------|
| 不知道發什麼 | 「給我選題」→ 視角庫 + `content-strategy` |
| 寫好了想更好 | 「幫我潤這篇」→ `copy-editing` + docs/02 預檢 |
| 貼文沒人看 | 「診斷這篇為什麼沒流量」→ docs/01 + `social` |
| 想收 email/LINE 名單 | 「設計名單磁鐵」→ `lead-magnets` |
| 想開始賣東西 | 「幫我設計 offer」→ `offers` + `pricing` |
| 數據看不懂 | 「本週成效總結」→ `analytics` + modules/2 |
| 完全沒靈感 | 「給我行銷點子」→ `marketing-ideas` |
