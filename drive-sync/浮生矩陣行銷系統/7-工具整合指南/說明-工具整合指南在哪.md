# 工具整合指南(tools/,1.2MB)取得方式

包含 51 個零依賴 Node.js CLI 工具、各行銷工具的 API 整合指南
(GA4、Stripe、Mailchimp、Resend…)、Composio 整合層,共 1.2MB。

## 取得完整內容(二選一)

1. **備份 zip**:`浮生矩陣行銷系統-完整備份.zip` → `7-工具整合指南/`
2. **GitHub**:https://github.com/xoxonco-ai/my-frist-project → `tools/` 目錄
   - `tools/REGISTRY.md` — 全部工具索引與能力對照
   - `tools/integrations/` — 每個工具的 API 端點、認證、常用操作
   - `tools/clis/` — 51 個 CLI 工具(Node 18+ 直接跑)

## 使用方式

這些工具是給 Claude 在 repo 裡執行用的,平常不需要手動翻:
需要接 GA4 / Stripe / 電子報時,直接跟 Claude 說
「幫我接 GA4 看數據」,它會自己去讀對應的整合指南。
