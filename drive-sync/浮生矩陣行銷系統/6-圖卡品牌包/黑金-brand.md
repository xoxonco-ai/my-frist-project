# Brand Pack · 浮生矩陣-黑金（貴氣・力量・洞察）

> social-cards-engine 品牌插件。用於**偏洞察/解碼/覺醒**的房間。
> 觸發：房間為 `見幻筆記`／`浮生解碼`／`登出指南`，或你說「黑金版」。
> 渲染實作：`render_preview.py`（theme=`blackgold`）。

## 🎨 視覺識別（近黑底・燙金・明體）
```css
--bg:#0b0a08;                    /* 近黑底 + 燙金放射/木紋質感 */
--ink:#f4efe6;                   /* 主字近白 */
--gold:#d8b24a;                  /* 燙金 accent：第二行標題 / 關鍵詞 */
--kicker:#c9a227;                /* 房間標籤金 */
--box-bg:rgba(0,0,0,.34);        /* 標題/內文框（半透明黑）*/
--box-bd:rgba(201,162,39,.38);   /* 金色細框 */
--sub:#e7dcc4;                   /* 內文暖白 */
```
- **字體**：明體/宋體 `Noto Serif TC`（免費商用，近思源宋體）— 貴氣、有重量。
- **簽名母題**：金色細框圓角卡 + 頂部單字水印（如「洞」）+ 頁尾 wordmark「浮生矩陣」+ `0X/07`。
- **背景**：低調燙金放射線 + 暗角，**不放模糊殘影文字層**（原 bug 來源，已移除）。
- **尺寸**：4:5（1080×1350），2× 高清。

## 🌿 敘事 DNA（共用）
- 第二人稱「**妳**」，溫柔不說教；`痛點對號 → 翻轉 → 一個小練習`。
- **anti-hype 硬邊界**（碰到即 carousel-joker FAIL）：不恐嚇／不神化／不絕對預測／不成功學／不顯化神話／不醫療療效宣稱。
- 黑金雖貴氣，**文字仍守溫柔**——用力量感的視覺承載溫柔的話，不讓視覺變威嚇。
