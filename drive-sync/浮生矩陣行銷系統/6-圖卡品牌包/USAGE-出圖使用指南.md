# 出輪播圖卡 · 使用指南（USAGE）

兩條路。要正式上稿 → 路 2（你 Mac）；想快速看樣 / 讓我批次跑 → 路 1（網頁）。

---

## 路 1️⃣ — 在網頁叫 Claude 生成（快速預覽）

在 **my-frist-project repo 開一個 Claude Code 網頁 session**，直接說：

> 出 D05 輪播卡

或更明確：

> 用 render_bg.py 出 D05（房間：回家）的 7 張輪播卡，CTA 關鍵字用「回家」

**Claude 會做的事**：讀當天內容 → 套對應主題（黑金/溫柔）→ 用 `render_bg.py` 出圖 → 傳圖給你。

**注意事項**
- 雲端環境是**臨時的**：程式碼在 git 裡（不會掉），但**底圖不進 git**（大檔）。
  換到新環境時 `浮生矩陣-黑金/backgrounds/` 會是空的 → 你要**重新上傳底圖**（拖進對話即可），
  或叫 Claude「把底圖也存進 repo」。
- 預覽字型是系統**黑體**，不是明體（明體要在你 Mac 上跑，見路 2）。

---

## 路 2️⃣ — 在你自己 Mac 上跑（正式上稿）✅ 推薦

用你本機的 `social-cards` skill + 做好的兩個 brand pack。

### 只需裝一次
```bash
# 把兩個 brand pack 複製進 social-cards-engine
cp -r 浮生矩陣-黑金 浮生矩陣-溫柔 <social-cards-engine>/brands/
# 底圖放進黑金 pack（你 Mac 上的底圖檔）
cp <你的底圖>/*.png <social-cards-engine>/brands/浮生矩陣-黑金/backgrounds/
```

### 之後每次出圖，在 Claude Code 說
> 用 social-cards 出 D05 圖卡，房間「回家」（→ 溫柔版），讀當天的 `2_1230_IG輪播貼文.md`，
> 依【輪播說明】做 7 張，第 7 張 CTA 關鍵字「回家」

**好處**：明體字、底圖在本機、跟你原本工作流一致、上稿級品質。
